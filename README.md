Proyecto: Análisis del Comportamiento del Consumidor en Retail
¡Hola! Este repositorio contiene un análisis de datos de extremo a extremo diseñado para entender las tendencias de compra de los clientes y optimizar las estrategias de negocio.

Herramientas Utilizadas
  Python (Pandas): Limpieza, manipulación y preparación de datos.
  SQL (PostgreSQL): Consultas avanzadas para análisis de negocio profundo.
  Power BI: Creación de un dashboard interactivo.
Pasos del Proyecto
  Limpieza de Datos (Python): Se cargó el conjunto de datos, se manejaron valores nulos imputando la mediana por categoría y se estructuraron las columnas para facilitar el análisis.
  Análisis de Negocio (SQL): Se realizaron consultas para responder preguntas clave como:
  Ingresos totales por género y categoría.
  Identificación de los productos más vendidos y mejor valorados.
  Análisis de la efectividad de los descuentos.
  Visualización (Power BI): Se construyó un tablero para visualizar KPIs clave de ventas y comportamiento del cliente, permitiendo filtrar por ubicación, género y tipo de suscripción.
